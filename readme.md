Space Legion TD
Game Concept — Competitive Tower Defense With Dual Attack/Defense
The game is a two-player competitive tower defense where both players defend and attack at the same time, inspired by Legion TD, but using classic fixed towers similar to Bloons TD instead of fighter units.
Each match is played through a sequence of standardized waves. Every wave sends a base set of enemies to both players. Players must build and upgrade towers, manage positioning, and optimize damage types to survive increasingly difficult waves.
However, the core competitive layer comes from player-controlled wave manipulation and offensive pressure.

Core Mechanics
1. Dual Economy (Defense vs Offense)
Players earn money after each wave and must constantly choose how to spend it:
Defense Investment


Build new towers


Upgrade existing towers


Improve synergies, coverage, and damage types


Prepare for future waves


Attack Investment


Send extra enemy units to opponent


Enhance strength of sent units


Modify upcoming wave composition


Apply pressure at key timing windows


Balancing greed vs safety is the heart of gameplay.

2. Wave Manipulation System
Players can influence the structure of the next wave:
Examples:
Increase ratio of tanks vs swarm units


Add fast units to break weak defenses


Add shielded / armored enemies


Boost wave speed or spawn density


Add a mini-boss modifier


This creates a mind-game layer where players try to exploit weaknesses in the opponent’s tower setup.

3. Standard Tower Defense Gameplay
Defense is built from traditional towers:
Fixed tower placement


Upgrade paths


Damage types & resistances


Range and positioning strategy


Area vs single-target optimization


Economy/greed towers vs pure defense


Good planning and layout matter as much as spending power.

4. Escalating Pressure
As waves progress:
Base waves become stronger


Offensive options unlock or scale


Players gain access to stronger attack modifiers


Mistakes become harder to recover from


Matches naturally build toward a breaking point where one player’s defense collapses.

Win Condition
A player loses when:
Too many enemies leak through


The opponent wins by successfully overwhelming their defense through better strategy, economy management, and wave pressure.

Gameplay Focus
The game emphasizes:
Strategic resource allocation


Reading opponent weaknesses


Timing offensive pressure


Tower synergy and layout optimization


Risk vs reward economy decisions
